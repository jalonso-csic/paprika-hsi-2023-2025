# paprika-hsi-2023-2025

\# Hyperspectral imaging of smoked paprika (VIS–NIR \& NIR–SWIR, 2023–2025)



\*\*Goal:\*\* Provide the MATLAB code and minimal materials to reproduce PCA, supervised models (PLS-DA/SVM with nested CV), and PLSR (moisture/ASTA) used in the manuscript.



\## Structure

\- `code-matlab/` — MATLAB scripts exactly as used in the study.

\- `data-sample/` — tiny sample dataset to verify the pipeline (not full data).

\- `docs/` — quick-start and workflow notes.

\- `figures/` — exported figures (small selection).

\- `supplementary/` — supplementary tables and run logs.



\## Quick start (5 min)

1\. MATLAB R2025a with \*\*Statistics and Machine Learning\*\* and \*\*Parallel Computing\*\* toolboxes.

2\. Add `code-matlab/` to the MATLAB path.

3\. Run the main script (to be added) and check outputs in `/figures` and `/supplementary`.
